# PMS UI (HTML/CSS/JS)

This is a page-wise UI implementation based on the Excel backlog (01_PBL_UI_PMS.xlsx).
It is a front-end demo only (no backend). Data is stored in the browser using localStorage.

## How to run
1. Extract the zip.
2. Open `pms-ui/index.html` in a browser (Chrome/Edge).
   - For best experience, use VS Code + Live Server.

## Seed accounts
- Customer: `customer01` / `Customer@123`
- Officer: `officer01` / `Officer@123`

## Pages
- Login
- Customer Registration
- Customer Home / Officer Home
- Booking Service (Customer / Officer)
- Pay Bill
- Invoice (print)
- Tracking (Customer / Officer)
- Pickup Scheduling (Officer)
- Delivery Update (Officer)
- Booking History (Customer / Officer)
- Customer Support
